# 임베디드 시스템의 종류

## 1. 임베디드 시스템의 정의

임베디드 시스템(Embedded System)은 특정 기능을 수행하기 위해 하드웨어와
소프트웨어가 하나의 장치에 내장된 컴퓨터 시스템이다. 일반 PC처럼 다양한
작업을 수행하는 것이 아니라 정해진 목적을 안정적이고 효율적으로
수행하도록 설계된다.

------------------------------------------------------------------------

## 2. 임베디드 시스템을 구성하는 최소 조건

-   하드웨어(CPU/MCU)
-   메모리(RAM, ROM, Flash)
-   입출력 장치(GPIO, 센서, 통신)
-   전원 공급 장치
-   소프트웨어(펌웨어 또는 운영체제)

------------------------------------------------------------------------

## 3. 펌웨어, RTOS, 임베디드 OS의 용도

### 펌웨어(Firmware)

-   하드웨어를 직접 제어하는 프로그램
-   Flash/ROM에 저장
-   전자레인지, 리모컨, 키보드 등에 사용

### RTOS(Real-Time Operating System)

-   실시간 처리가 필요한 시스템에서 사용
-   일정 시간 안에 반드시 작업 수행
-   자동차 ECU, 산업용 로봇, 드론 등

### 임베디드 OS

-   다양한 기능 제공
-   멀티태스킹, 파일 시스템, 네트워크 지원
-   Linux, Android, Windows IoT 등

## 6. 새로운 임베디드 시스템 아이디어

분리수거하는 시스템

-   모터
-   LCD
-   Wi-Fi
