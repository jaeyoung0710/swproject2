# 아날로그 센서?
- 연속적인 전압값으로 데이터를 출력하는 센서이다 아두이노는 ADC를 사용하여 아날로그값을 디지털 수치로 변환한다.

# 디지털 센서?
- 처음부터 0과 1 또는 디지털 데이터 형태로 값을 출력하는 센서이다. 

# 물리 값을 측정해주는 센서
- 온도센서: 온도를 측정한다.
- 습도센서: 공기 중 습도를 측정한다.
- 조도센서: 빛의 밝기를 측정한다.
- 초음파센서: 거리를 측정한다.
- 가속도센서: 움직임과 기울기를 측정한다.
- 압력센서: 압력이나 무게를 측정한다.
- 가스센서: 가스 농도를 측정한다.
- 수위센서: 물의 높이를 측정한다.

# 엑추에이터란?
- 엑추에이터는 컴퓨터의 출력장치를 말한다. 
- 빛을 이용하는 장치
    - LED
    - LCD
    - OLED
    - 7세그먼트
    - 경고등 등등
- 모터 기반 액추에이터
    - DC 모터
    - 서브 모터
    - 진동 모터
    - 팬 모터

# 두개의 조합에 따라 만들수 있는 실제 인베디드 시스템
- 조도센서 + LED
→ 자동 가로등
- 온도센서 + 팬
→ 자동 냉각 장치
- 습도센서 + 가습기
→ 자동 습도 조절기
- 초음파센서 + 부저
→ 주차 거리 경고 장치
- PIR 센서 + 경고등
→ 보안 감지 시스템
- 수위센서 + 펌프
→ 자동 급수 장치

# 실생활에 사용되고 있는 임베디드 장비
- 스마트 도어락
    - 센서: 키패드, RFID 센서, 지문 센서
    - 액추에이터: 모터, 부저, LED

- 로봇청소기
    - 센서: 초음파센서, 충돌센서, 자이로센서
    - 액추에이터: 바퀴 모터, 흡입 모터, 브러시 모터
