# 아두이노에서 사용되고 있는 마이크로 콘트롤러의 종류와 성능 비교
- ATmega328P : 가장 기본적인 8비트 MCU로 소규모 프로젝트와 교육용에 적합하다.
- ATmega2560 : 8비트이지만 메모리와 입출력 핀이 많아 복잡한 제어 시스템에 적합하다.
- ATSAM3X8E : 32비트 Cortex-M3 기반으로 연산 성능과 메모리 용량이 뛰어나 고성능 임베디드 시스템에 적합하다.
- SAMD21 : 32비트 Cortex-M0+ 기반으로 저전력 특성이 우수하며 IoT 및 배터리 기반 장비에 적합하다.

# 아두이노로 220V 교류 전원을 사용하는 장비를 제어하는 방법?
: 릴레이 모듈을 사용하여 아두이노의 디지털 출력 핀으로 릴레이를 키고 끄면 릴레이가 스위치 역할을 하여 220V 전등, 선풍기, 히터 같은 장비의 전원을 on/off 할수 있다.

# 아두이노에서 동작하는 펌웨어 소프트웨어의 특징을 설명하라
: 아두이노에서 동작하는 펌웨어는 C/C++ 언어로 작성되며 컴파일 후 마이크로 컨트롤러의 flash 메모리에 저장되어 실행됩니다. 또한 아두이노에는 부트로더가 내장되어 있어 usb 케이블 만으로 프로그램을 업로드 할 수 있고 별도의 프로그램 기록 장치 없이도 쉽게 개발 할 수 있다.

# RTOS 및 임베디드 리눅스와의 차이점을 비교 조사하라
- RTOS는 실시간 운영체제로 정해진 시간 안에 작업을 반드시 처리해야 하는 시스템에서 사용됩니다. 여러 작업을 동시에 할 수 있으며, 응답 속도가 매우 빠르다 
- 임베디드 Linux는 다양한 기능을 제공하는 운영체제로, 멀티 태스킹, 파일 시스템, 네트워크 기능 등을 지원합니다. 메모리 사용량은 많지만 복잡한 프로그램을 실행할 수 있다는 특징이 있습니다.
- 아두이노 펌웨어는 운영체제 없이 단일 프로그램이 실행되는 구조이므로 메모리 사용량이 적고 제어가 단순한 장치에 적합합니다. 따라서 펌웨어는 단순한 제어, RTOS는 실시간 제어, 임베디드 Linux는 복잡한 기능과 멀티테스킹이 필요한 시스템에 적합합니다.

# 아두이노의 전력 소비량 추정, 배터리를 얼마나 버틸 수 있을지 계산해보자
- 아두이노 소비전력을 확인하기 위해 아무런 기능을 수행하지 않는 빈 스케치를 업로드하고 전류를 측정하였을때 아두이노 우노 R3은 5.08V 전원에서 약 64.3mA의 전류를 소비하고 같은 보드를 4.0 배터리 전원으로 동작하였을 때는 42.0mA로 소비 전류가 감소하였습니다.따라서 배터리 사용 시간을 계산하기 위해 2000mAh 배터리를 사용한다고 가정하면 64.3mA 소비시 20000mAh/64.3mA = 대략 31시간 42.0mA 소비시 대략 48시간 정도 사용할 수 있습니다.
