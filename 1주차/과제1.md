# 임베디드 시스템이란?
- 임베디드 시스템은 특정 기능을 수행하기 위한 하드웨어가 하나의 장치에 내장된 컴퓨터 시스템이다

# 임베디드 시스템을 구성하는 최소 조건은?
- 하드웨어(CPU.MCU)
- 메모리(RAM, ROM, Flash)
- 입출력 장치(GPIO, 센서, 통신)
- 전원 공급 장치
- 소프트웨어(펌웨어 또는 운영체제)

# 펌웨어, RTOS, 임베디드 OS의 용도는?
- 펌웨어: 하드웨어를 직접 제어
- RTOS: 실시간 처리가 필요한 시스템에서 사용
- 임베디드 OS : 다양한 기능 제공, 멀티태스킹, 파일시스템, 네트워크 지원

# 임베디드 시스템의 종류
- 차량용
: ECU, ABS, 자율주행 제어장치, 네비게이션 등등
- 산업용
: PLC, 산업용 로봇, 공장 자동화 시스템 등등
- 의료용
: MRI, CT, 심박수 측정기, 인공호흡기 등등
- 가정용
: 전자레인지, 세탁기, 스마트 냉장고 등등
- 인터넷 장비
: 공유기, 스위치 등등
- 로봇
: 산업용 로봇, 서비스 로봇, 자율주행 로봇 등등
- 우주항공 분야
: 인공위성, 항공기 비행 제어 시스템, 드론
- 그외
: 키오스크, ATM, 스마트폰 등등

# 임베디드 시스템 사례
- 통신용 네트워크 장비
- 산업용 로봇
- 배송용 드론
- 스마트 CCTV
- 스마트 도어락

## 보너스 과제
# 새롭게 만들고자 하는 임베디드 시스템?
- AI 냉장고 식재료 관리기
    - 냉장고 안의 식재료를 인식하여 유통기한을 관리하여 유통기한이 얼마남지 않은 식재료를 알려주고 미리 품목을 저장해두면 식재료가 다 떨어져가면 구매가 필요한 식재료를 알려주는 임베디드 시스템
    - 구현에 필요한 구체적 요소
        - 하드웨어 : 카메라, 무게센서, 온도센서 등등 
        - 소프트웨어: 식재료 인식 AI 프로그램, 유통기한 관리 프로그램, 스마트폰과 연동하여 알림을 보내는 프로그램 등등
